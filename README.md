# Sistema Web para Restaurante

Proyecto universitario desarrollado utilizando **Python** y **Flask**, cuyo objetivo es ofrecer una plataforma web para la gestión y visualización de información de un restaurante. La aplicación permite a los usuarios consultar el menú, conocer información del establecimiento y navegar por una interfaz web moderna y amigable.

---

## 📖 Descripción

Este proyecto fue realizado como parte de una asignatura universitaria orientada al desarrollo web con Python. Se implementó utilizando el framework Flask siguiendo una estructura organizada basada en plantillas HTML, archivos estáticos y rutas dinámicas.

---

## 🚀 Tecnologías Utilizadas

- Python 
- Flask
- HTML
- CSS
